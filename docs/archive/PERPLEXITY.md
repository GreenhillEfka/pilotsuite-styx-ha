# Perplexity (direct) on-demand

Two modes are available:

## Quick (default)
- Model: `sonar-pro`
- Trigger: `pplx:` or `perplexity:`

## Deep
- Model: `sonar-reasoning-pro`
- Trigger: `pplx-deep:`

Scripts:
- `pplx-quick.mjs` — quick mode
- `pplx-deep.mjs` — deep mode

Note: These are invoked by the assistant via exec when you use the triggers.
